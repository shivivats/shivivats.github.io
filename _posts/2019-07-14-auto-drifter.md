Auto Drifter is a **top-down 2D racing** game in which the player competes against the AI in a race, drifting around corners while the car automatically accelerates. The game was made for a course as part of my [Masters degree](https://www.aau.at/en/studien/master-game-studies-and-engineering/), with the condition that the game had to only be played with one button. I made two versions of the game, one played with one button where the player can only turn counter-clockwise, and another where the player can turn either direction using two buttons.

The game was built using the **Unity3D** engine and written in **C#**. I worked alone on this, hence I programmed the **gameplay** and **created the art**. The game does not have any audio.
