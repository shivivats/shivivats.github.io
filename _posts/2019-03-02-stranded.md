Stranded is a **2D platformer** where the player slings themselves around planets to get to their spaceship. It was developed for [Blackthornprod Game Jam #2](https://itch.io/jam/blackthornprodgamejam2). The game contains 3 total levels, each with varying level of difficulty, with the first planet acting as a tutorial level as well.

The game was built using the **Unity3D** engine and written in **C#**. I worked in a team of five on this, as a **gameplay programmer**, including the **slinging functionality** which was inspired by the bash functionality in Ori and the Blind Forest.
